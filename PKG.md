# External Libraries

Categorized list of external dependencies used in `skillzeug`.

## CLI Framework
- [Cobra](https://github.com/spf13/cobra): A library for creating powerful modern CLI applications.
- [Pflag](https://github.com/spf13/pflag): Drop-in replacement for Go's flag package, implementing POSIX/GNU-style --flags.

## Terminal User Interface (TUI)
- [Bubble Tea](https://github.com/charmbracelet/bubbletea): A powerful little TUI framework based on the Elm Architecture.
- [Bubbles](https://github.com/charmbracelet/bubbles): Common TUI components for Bubble Tea.
- [Lip Gloss](https://github.com/charmbracelet/lipgloss): Style, layout, and color definitions for TUI applications.

## Terminal & Character Handling
- [Termenv](https://github.com/muesli/termenv): Advanced ANSI style & color support for the terminal.
- [Go-isatty](https://github.com/mattn/go-isatty): Implementation of `isatty` for Go.
- [Go-runewidth](https://github.com/mattn/go-runewidth): Functions to get fixed width of the character or string.
- [Uniseg](https://github.com/rivo/uniseg): Unicode Text Segmentation for Go.
- [Ansi](https://github.com/charmbracelet/x/ansi): ANSI escape sequence parser and generator.
- [Go-colorful](https://github.com/lucasb-eyer/go-colorful): Library to play with colors in Go.
- [UAX-29](https://github.com/clipperhouse/uax29): Unicode Text Segmentation (UAX #29) for Go.

## System & Utilities
- [Go-sys](https://golang.org/x/sys): Go packages for low-level interaction with the operating system.
- [Go-text](https://golang.org/x/text): Go packages for text processing.
- [Clipboard](https://github.com/atotto/clipboard): A Go library to provide copying and pasting to the Clipboard.
- [Mousetrap](https://github.com/inconshreveable/mousetrap): Detect if a Go application was started by a double-click on Windows.
- [Cancelreader](https://github.com/muesli/cancelreader): Library to cancel reader operations.
